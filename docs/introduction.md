# ForgePulse

> **Note:** This project was formerly known as **FlowForge**. It has been renamed to **ForgePulse**.

A powerful, production-ready Laravel 12 package that provides a visual drag-and-drop workflow builder with conditional branching, real-time execution tracking, and comprehensive integration capabilities.

## Key Features

- **Laravel 12** - Built for the latest Laravel version
- **Livewire 4** - Real-time reactive components
- **PHP 8.3-8.5** - Modern PHP support
- **Production Ready** - Battle-tested and reliable
