FlowForge

A powerful, production-ready Laravel 12 package that provides a visual drag-and-drop workflow builder with conditional branching, real-time execution tracking, and comprehensive integration capabilities.

Laravel 12 Livewire 4 PHP 8.3-8.5 Production Ready

Drag & Drop Builder

Visual workflow designer with real-time updates using Livewire 4

Conditional Logic

Complex boolean expressions with 15+ comparison operators

Real-Time Tracking

Live execution monitoring with performance metrics

Multi-Language

Built-in support for 5 languages with RTL

Template System

Save, load, and share workflow configurations

Secure & Tested

Policy-based authorization with comprehensive tests

Installation

Get started with FlowForge in minutes using Composer.

Install the package using Composer:

Publish the configuration file to customize FlowForge:

php artisan vendor:publish --tag=flowforge-config

This creates config/flowforge.php where you can configure execution settings, permissions, and more.

üóÑÔ∏è Step 3: Run Migrations

Publish and run the database migrations:

php artisan vendor:publish --tag=flowforge-migrations
php artisan migrate

This creates 4 tables: workflows, workflow_steps, workflow_executions, and workflow_execution_logs.

üé® Step 4: Publish Assets (Optional)

Optionally publish views, assets, and language files:

php artisan vendor:publish --tag=flowforge-views

# Publish CSS/JS assets
php artisan vendor:publish --tag=flowforge-assets

# Publish language files
php artisan vendor:publish --tag=flowforge-lang
All set! You're ready to create your first workflow. Continue to the Quick Start section.

Quick Start

Create your first workflow in under 5 minutes.

1. Create a Workflow

use AlizHarb\FlowForge\Models\Workflow;
use AlizHarb\FlowForge\Enums\WorkflowStatus;

$workflow = Workflow::create([
    'name' => 'User Onboarding',
    'description' => 'Automated user onboarding process',
    'status' => WorkflowStatus::ACTIVE,
]);

2. Add Steps

use AlizHarb\FlowForge\Enums\StepType;

// Send welcome email
$workflow->steps()->create([
    'name' => 'Send Welcome Email',
    'type' => StepType::NOTIFICATION,
    'position' => 1,
    'configuration' => [
        'notification_class' => \App\Notifications\WelcomeEmail::class,
        'recipients' => ['{{user_id}}'],
    ],
]);

// Wait 24 hours
$workflow->steps()->create([
    'name' => 'Wait 24 Hours',
    'type' => StepType::DELAY,
    'position' => 2,
    'configuration' => [
        'seconds' => 86400,
    ],
]);

3. Execute the Workflow

// Execute asynchronously (queued)
$execution = $workflow->execute([
    'user_id' => $user->id,
    'email' => $user->email,
]);

// Execute synchronously
$execution = $workflow->execute(['user_id' => $user->id], async: false);

4. Use the Visual Builder

<livewire:flowforge::workflow-builder :workflow="$workflow" />
